this document list the requirement and a simple prototype

MTV(Model/Template/View)

Model:
    book
        Name
        dateOfPublish
        author (M to M)
    author
        Name
        Email
    user
        Name
        Email
    review
        user
        book
        dateOfRead
        reviewOfScore
        comment
    find book/find author
    average review of score for each book
    total review number for each book
    sort book by name, author
    sort book by average review of score/total review number for each book

View:
    list view: list all books, or list books with keyword, sort by different field
    detail book view: detail of book, could add/edit/delete/review a book
    detail author view: detail of author
    detail user view: detail of user
